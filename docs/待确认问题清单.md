# 待确认问题清单

以下问题必须在拿到官方文档后尽快确认。

## 评分与目标

- 最终评分是纯收益，还是收益与预测误差结合
- 预测目标是单点价格、价格序列，还是多时段联合输出
- 是否有额外惩罚项
- 是否有线上榜单与线下最终榜单差异

## 数据与标签

- 时间粒度是否固定为 15 分钟
- 训练集、验证集、测试集的时间范围
- 是否提供节点标识、地理相关信息或网络相关字段
- 是否包含负荷、新能源、气象、市场交易等全部公开描述中的字段
- 缺失值与异常值处理要求
- 时间戳是否统一到同一时区

## 储能约束

- 容量、功率、效率的具体定义
- 初始 SoC 与终端 SoC 是否固定
- 是否允许同时充放电
- 是否有循环次数、吞吐量或退化约束
- 是否有最小充放电时长
- 是否有限价、功率分段、市场参与规则等附加约束

## 提交与工程

- 提交的是结果文件还是 Docker 镜像
- 输入输出目录约定
- 运行时长限制
- 显存、内存、CPU 限制
- 是否允许联网
- 依赖安装方式
- 每天提交次数限制

## 评测逻辑

- 是滚动预测还是一次性批量预测
- 是否在线模拟未来逐步揭示
- 收益是否按全周期累计
- 是否存在隐藏测试集与公开测试集分布差异
