# 给项目经理的首次对齐提纲

## 本次对齐目标

- 把赛题理解对齐
- 把分工对齐
- 把接下来 7 天的动作对齐

## 建议直接说的版本

这个题本质上不是单纯做电价预测，而是“节点实时电价预测 + 储能充放电优化 + 工程提交”。我们第一阶段不追复杂模型，先做出一个无泄漏、可回测、可解释、可提交的 baseline，再请山东那边帮我们校准业务假设。

## 建议确认的事项

- 报名与官方文档获取由谁跟
- 与山东支持方沟通由谁约
- 每天或每两天同步一次，固定用同一个模板
- 文档、结果和问题都落在当前目录里，避免信息散落

## 每次同步固定模板

- 本轮目标：
- 当前 baseline：
- 当前最优收益：
- 与 oracle 差距：
- 本轮改动：
- 风险与待确认：
- 下一步动作：

## PM 侧重点

- 跟踪官方文档
- 维护问题清单
- 推动外部业务访谈
- 记录业务假设和修正结论
- 控节奏，不让团队过早发散到复杂模型
