# 山东业务访谈提纲

## 访谈目标

- 用最短时间校准团队对电力市场和储能业务的理解
- 找到最可能影响收益的关键业务因素
- 避免团队在错误假设上投入过多建模时间

## 开场说明

我们正在准备“电力市场交易赛道（储能电站收益优化）”，团队算法和工程能力强，但对电力市场业务理解有限。希望先确认几个关键业务点，避免后续建模方向跑偏。

## 核心问题

- 蒙西实时价格最主要的驱动因素有哪些，能否按重要性排一下
- 价格尖峰通常由什么触发
- 节点价差更多受什么影响，新能源波动还是网络拥塞
- 做储能收益优化时，最容易漏掉的业务约束是什么
- 如果只做第一版 baseline，最应该先关注哪些特征
- 在实务中，抓高价尖峰更重要，还是抓低价和负价时段更重要
- 对储能而言，哪些“看上去有收益”的策略实际在业务上不可行
- 没有电力背景的算法团队最常见的误区有哪些

## 如果对方愿意继续深入，可追加的问题

- 是否存在明显的季节性或天气驱动模式
- 负荷、风电、光伏各自对价格波动的贡献大致如何
- 市场规则层面有哪些会直接影响建模设计
- 如果只能给我们一个建议，最值得优先补的业务知识是什么

- 对方的经验判断
- 需要查官方规则验证的点
- 需要修正的团队业务假设
